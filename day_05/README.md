# 🎵 イントロドン！

ジャンル選択でも、アーティスト検索でも遊べるイントロクイズゲームです。  
iTunes Store の30秒プレビューを使って、曲名を当てる形式になっています。

## できること

- ジャンル選択でイントロドン
- ランダムジャンルでイントロドン
- アーティスト名で検索して候補から選択
- 選んだアーティストの曲だけでイントロドン
- 5問 / 10問 / 15問の問題数切り替え
- 結果画面から `Xでシェア`
- 結果画面から `URLをコピー`

## 遊び方

1. タイトル画面で `ジャンルで遊ぶ` か `アーティストで遊ぶ` を選ぶ
2. ジャンルを選ぶ、またはアーティストを検索して1組選ぶ
3. 問題数を選んでゲーム開始
4. プレビューを聴いて曲名を当てる
5. 結果画面からスコアをシェアする

## スコア

| ヒント | 再生時間 | 得点 |
| --- | --- | --- |
| Lv.1 | 1秒 | 500pt |
| Lv.2 | 3秒 | 400pt |
| Lv.3 | 5秒 | 300pt |
| Lv.4 | 10秒 | 200pt |
| Lv.5 | 30秒 | 100pt |

## 使用API

- iTunes Search API
  - ジャンル検索
  - アーティスト検索
  - アーティストごとの楽曲プレビュー取得

## ローカルで確認する方法

`file://` で直接開くと API の制約で動かないことがあるため、HTTP サーバー経由で確認してください。

```bash
# Node.js がある場合
npx serve .

# Python がある場合
python -m http.server 8080
```

起動後、`http://localhost:8080/day_05/` にアクセスします。
